Workbooks, worksheets, and spreadsheets

Photo by Rooty

The Microsoft Excel spreadsheet file format is widely used for storing tabular data. The spreadsheet structure is based on a grid of cells with numbered rows and letter-named columns. Although the software provides many additional features such as macros, formulas, charts, and pivot tables, the ingest solutions are focused on the use of Excel as a data source. An Excel spreadsheet can be exported as a delimited data file as an alternative option.

  • The most common file formats are the binary (*.xls) and xml (.xlsx) workbook formats.
  • Workbook files enable the storage of data in multiple sheets.
  • Each sheet contains a set of data equivalent to a table.
  • Unlike the conventions of a delimited data file, column headers may not be in the first row.
  • A worksheet can contain merged columns, rows, and unstructured text.
  • The smallest unit of stored data is a cell, the intersection of a row and column.
